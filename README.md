## Description

This cross-platform mobile application, developed with React Native, delivers a modern and fluid tracking dashboard for cryptocurrency enthusiasts. The app enables users to securely view a real-time list of cryptocurrencies, inspect detailed asset metrics, and persist their favorite tokens to a personalized watchlist ("My Exchange"). It leverages Firebase to handle efficient data persistence and retrieval seamlessly, ensuring a responsive and synchronized user experience. Built using declarative component principles, structured state management, and a native tab bar navigation system, this project showcases optimal workflows for financial data tracking in a mobile environment.

## Features

- **React Native**: Used for building a high-performance cross-platform mobile app using JavaScript/TypeScript.
- **Firebase**: Utilized as the backend service to store and fetch real-time cryptocurrency data dynamically.
- **Tab Navigation**: Offers a clean separation between the live market dashboard and the user's watchlist.
- **State Management**: Implements efficient data flow hooks to propagate real-time model updates instantly across screens.
- **Watchlist Persistence**: Enables users to seamlessly bookmark assets with "Add to Favorites" and manage allocations cleanly with an all-inclusive "Remove All" command.
- **Component Architecture**: Demonstrates modular, reusable component patterns to keep presentation and data logic separate and maintainable.
